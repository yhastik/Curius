Основы HTML

HTML (HyperText Markup Language) — это язык разметки для создания веб-страниц. Он используется для структурирования контента.

Основные теги HTML:

<html>: Корневой элемент.
<head>: Метаинформация (заголовок, стили, скрипты).
<title>: Заголовок страницы.
<body>: Основной контент.
<h1>-<h6>: Заголовки.
<p>: Параграф.
<a>: Ссылка.
<img>: Изображение.
<ul>, <ol>, <li>: Списки.
Базовая структура HTML:

<!DOCTYPE html>
<html lang="ru">
<head>
<meta charset="UTF-8">
     <meta name="viewport" content="width=device-width, initial-scale=1.0">
     <title>Моя страница</title>
</head>
<body>
     <h1>Добро пожаловать!</h1>
     <p>Пример текста.</p>
</body>
</html>

Семантические теги

Семантические теги:

<header>: Заголовок страницы/раздела.
<footer>: Нижний колонтитул.
<article>: Самостоятельный контент.
<section>: Раздел документа.
<nav>: Навигация.
<aside>: Дополнительный контент.
премер:

<header>
     <h1>Заголовок</h1>
     <nav>
          <ul>
               <li><a href="">Главная</a></li>
               <li><a href="">О нас</a></li>
          </ul>
     </nav>
</header>
<main>
     <article>
          <h2>Статья</h2>
          <p>Текст...</p>
     </article>
</main>
<footer>
     <p>Футер</p>
</footer>

Формы

Теги форм:

<form>: Контейнер для формы.
<input>: Поле ввода.
<textarea>: Многострочный ввод.
<select>: Выпадающий список.
премер:

<form action="/submit" method="post">
     <label for="name">Имя:</label>
     <input type="text" id="name" name="name" required>
     <br>
     <label for="email">Email:</label>
     <input type="email" id="email" name="email" required>
     <br>
     <input type="submit" value="Отправить">
</form>

Валидация форм

Теги для валидация форм:

required: Обязательное поле.
pattern: Регулярное выражение для проверки.
minlength: Минимальная длина ввода.
премер:

<input type="text" id="username" name="username" required pattern="[A-Za-z]{3,}" title="Не менее 3 букв">
