Основы CSS

CSS (Cascading Style Sheets) — язык стилей для оформления HTML-документов. Он разделяет содержание и оформление, делая страницы более управляемыми и адаптивными.
Стилизация веб-страниц

Селекторы, свойства, значения

Селекторы: Выбирают элементы для стилизации.

Типовые: p { color: blue; } (все <p>).
Классы: .classname { color: red; } (элементы с классом).
ID: #idname { color: green; } (элемент с ID).
Атрибуты: [type="text"] { color: purple; }.
Псевдоклассы: a:hover { color: yellow; }.
Свойства: Определяют аспекты стиля (например, color, font-size).
Значения: Задают параметры стиля (например, red, 16px).
пример:

p {
     color: blue;
     font-size: 16px;
}

Внедрение CSS

1. Встроенные стили: Через атрибут style.

<p style="color: blue;">Текст</p>

2. Внутренние стили: Внутри <style> в <head>.

<style>
     p { color: blue; }
</style>
3. Внешние стили: В отдельном файле, подключаемом через <link>.

<link rel="stylesheet" href="styles.css">

Модель коробки (Box Model)

Content: Содержимое.
Padding: Внутренние отступы.
Border: Граница.
Margin: Внешние отступы.
пример:

.box {
     width: 200px;
     padding: 20px;
     border: 5px solid black;
     margin: 30px;
}

Раскладка и позиционирование

Flexbox

Гибкая модель для выравнивания элементов.

.container {
     display: flex;
     justify-content: space-between;
     align-items: center;
}
Grid

Двухмерная раскладка.

.container {
     display: grid;
     grid-template-columns: repeat(3, 1fr);
     gap: 10px;
}

Позиционирование

position:

static (по умолчанию).
relative (относительно себя).
absolute (относительно предка).
fixed (относительно окна).
sticky (гибрид relative и fixed).
float: Позволяет элементу "плавать".
.element {
     position: absolute;
     top: 10px;
     left: 20px;
}

Адаптивный дизайн и медиазапросы

Медиазапросы позволяют адаптировать стили под разные устройства.
@media (max-width: 600px) {
     .container {
               flex-direction: column;
     }
}
@media (min-width: 601px) and (max-width: 1200px) {
     .container {
          flex-direction: row;
     }
}
@media (min-width: 1201px) {
     .container {
          flex-direction: row;
     }
}
